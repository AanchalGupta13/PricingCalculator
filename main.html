About AWS Pricing Calculator

A web-based Pricing Calculator platform designed to streamline AWS infrastructure cost estimation. Users can effortlessly upload configuration files through the interface, once it processed the file, users can seamlessly download the generated pricing report directly from the website. The platform also features an intelligent Price BOT, allowing users to input server, storage, and database configuration requirements to provide accurate, real-time AWS cost estimations.

Features

  • Price BOT supports multi server queries
  • Real-time pricing calculations
  • Secure file storage
  • Supports Dual Input Modes- File Upload/Download Section: Process Excel/CSV configuration sheets & Query Interface Price BOT: Describe server requirements in plain English

How to Use

  • Upload your server configuration files (only Excel or csv format)
  • Select your result file and download
  • Use the Price BOT and can query like -
    - 24 Cores CPU, 128 GB RAM, 512GB SSD storage and Microsoft SQL Server database
    - 24 Core CPU, 128 GB RAM, 512GB SSD storage
    - 24 vCPU, 128 GB RAM, 512GB SSD storage
